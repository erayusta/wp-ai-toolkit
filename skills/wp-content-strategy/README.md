# WP Content Strategy

> Plan topic clusters, keyword research, and content calendars for WordPress sites

## What This Skill Covers

- Building topic cluster architectures with pillar and supporting content
- Conducting keyword research and mapping terms to content types
- Creating editorial content calendars with publishing schedules
- Analyzing content gaps and competitor positioning

## Example Usage

Ask your AI assistant:

```
> "Build a topic cluster strategy for a fitness blog on wordpress.test"
```

### Example Output

```
Topic Cluster: "Home Workout Routines"

Pillar Page: /home-workout-guide/
  Target keyword: "home workout routines" (2,400 mo/search)
  Word count: 3,000-4,000

Supporting Content:
  1. /home-workout-guide/bodyweight-exercises/
     Keyword: "bodyweight exercises at home" (1,800 mo/search)
     Internal link to: Pillar page, Post #3

  2. /home-workout-guide/no-equipment-cardio/
     Keyword: "cardio without equipment" (1,200 mo/search)
     Internal link to: Pillar page, Post #1

  3. /home-workout-guide/resistance-band-training/
     Keyword: "resistance band workouts" (900 mo/search)
     Internal link to: Pillar page, Post #1

Content Calendar:
  Week 1: Publish pillar page
  Week 2: Post #1 (bodyweight exercises)
  Week 3: Post #2 (no-equipment cardio)
  Week 4: Post #3 (resistance bands)
  Week 5: Update pillar page with internal links
```
